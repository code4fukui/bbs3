# bbs3 for Internet3

Internet3ユーザー専用のBBS（電子掲示板）アプリケーションです。

## 機能
- BBSへのメッセージ投稿
- 他のユーザーが投稿したメッセージの閲覧
- 自分が投稿したメッセージの削除

## 要件
- [Deno](https://deno.land/) 実行環境

## 使い方
1. リポジトリをクローンします:
```
git clone https://github.com/code4fukui/bbs3.git
```
2. プロジェクトディレクトリに移動し、アプリケーションを実行します:
```
deno run -A bbs3.js
```
BBSには、ご自身のEVER/IPアドレスと指定したポート（デフォルトは8001）でアクセスできます。
