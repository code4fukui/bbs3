# bbs3 for Internet3

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A BBS (Bulletin Board System) application for Internet3 users only.

## Features
- Post messages to the BBS
- View messages posted by other users
- Delete your own messages

## Requirements
- [Deno](https://deno.land/) runtime environment

## Usage
1. Clone the repository:
```
git clone https://github.com/code4fukui/bbs3.git
```
2. Navigate to the project directory and run the application:
```
deno run -A bbs3.js
```
The BBS will be accessible at your EVER/IP address and the specified port (default is 8001).
